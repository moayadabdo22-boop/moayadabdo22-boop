## 🔥 Featured Projects
 
### 🏋️ [FitCoach AI](https://github.com/moayadabdo22-boop/FitCoach-AI) — Intelligent Virtual Personal Trainer
`React` `FastAPI` `RAG` `FAISS` `LLMs` `scikit-learn` `Python`
 
Graduation project at Zarqa University: a bilingual (Arabic/English) AI fitness platform. Generates personalised workout and nutrition plans via a RAG pipeline, analyses meal photos and medical reports with LLMs, and includes an interactive muscle-map browser plus an admin dashboard for coaches and doctors.
 
### 🐱🐶 [Cat vs Dog Classification](https://github.com/moayadabdo22-boop/cat-dog-classification) — Hybrid Deep Learning
`PyTorch` `EfficientNet-B0` `OpenCV` `Python`
 
Three-phase framework on 10,000+ images combining a fine-tuned EfficientNet-B0 with classical CV (Harris Corner Detection, ORB, Haar Cascade) for real-time annotated inference. Reached 93.5% accuracy and 0.9354 macro F1.
 
### 🔐 [Family Vault](https://github.com/moayadabdo22-boop/family-vault-android) — Android Password & Notes Manager
`Java` `Android` `Fragments` `ViewPager2` `Material Design`
 
Native Android app that lets families keep household passwords, PINs, and notes in one private, offline place. Fragment-based navigation with Safe Args, tap-to-reveal password masking, and local storage via SharedPreferences.
 
### 🏠 [House Price Prediction API](https://github.com/moayadabdo22-boop/ML_Deployment)
`FastAPI` `scikit-learn` `Python`
 
Production ML REST API built during my internship at the Ministry of Digital Economy & Entrepreneurship — predicts California house prices from real-world feature sets, dataset to live endpoint.
